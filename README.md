# Calculator App

A simple calculator built using HTML, CSS, and JavaScript.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear button
* Saves calculation using Local Storage

## Technologies Used

* HTML
* CSS
* JavaScript

## Live Demo

 https://cse23r01a05cq.github.io/Calculator/


## How It Works

* Users click buttons to enter numbers and operators.
* JavaScript stores the calculation in a variable.
* `eval()` is used to calculate the result.
* Local Storage saves the calculation even after refreshing the page.
